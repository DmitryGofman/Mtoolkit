# Chapter 13 visuals manifest — carpentry & wood (נגרוּת)

All images sourced from Wikimedia Commons only. License verified via the Commons API
(`action=query&prop=imageinfo&iiprop=extmetadata`) before download. Files saved to
`app/public/visuals/` with the `ch13-` prefix. Every file was visually inspected
after download (not just trusted by filename/caption) to confirm it actually shows
what its Commons description claims. Oversized photos were re-encoded (resized to
1000px long edge + re-compressed JPEG) to stay under the ~400KB budget; noted per
file below. The Commons API rate-limited this session repeatedly (HTTP 429) —
lookups were paced with delays and retry/backoff, no license checks were skipped
to work around it.

## Gaps / honest notes

- **Topic 6 (warping) is a CGI render, not a real photograph.** I searched
  extensively for a genuine photo of a warped/cupped/twisted real wood board —
  English terms ("warped wood board", "cupped board timber", "bowed lumber
  plank", "twisted wood board", "warped floorboard", "warped door wood",
  "board cupping") and German ("Holzverzug", "verzogenes Holz", "gewölbtes
  Brett") — and checked Commons' `Category:Wood defects` and its subcategories.
  Real photographs of moisture-induced warping essentially don't exist as
  standalone, clearly-captioned Commons uploads (the category tree covers rot,
  insect damage, knots, and tree-growth curvature like "Säbelwuchs", not
  board-level cupping/bowing). The 3D render used is honest and pedagogically
  clear, but it is a diagram, not documentary evidence — flagged accordingly in
  its entry above.
- **Topic 4 (classic wood joints) covers only dovetail, not mortise-and-tenon or
  finger joints.** Good alternates were found and license-checked but not
  downloaded (to keep one image per topic): `File:Mortise and tenon joint.svg`
  (CC BY-SA 3.0, clean SVG diagram) and `File:Tenon and Mortise (PSF).png`
  (Public domain, Pearson Scott Foresman line drawing) both passed the license
  check and are available as a second joint-family image if wanted.
- **Topic 8 also had a plainer alternative**, `File:Woodworking-joint-butt-dowel.gif`
  (CC BY-SA 3.0, a simple flat diagram of a dowel-reinforced butt joint) — not
  used since the 3D-rendered `Dübelverbindung.jpg` shows the actual dowel
  hardware more clearly.
- **Topic 5** could equally have used a CNC router shot instead of the table saw
  (`File:CNC wood router typical work.JPG`, CC BY-SA 4.0, Doug Coldwell — passed
  license check, not downloaded) if a CNC-specific image is later preferred over
  the table saw action shot.
