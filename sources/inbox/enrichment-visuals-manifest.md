## ch16-vernier-scale-reading.gif

**What it shows:** The classic English-Wikipedia-style "how to read a vernier scale"
diagram: a yellow vernier (nonius) scale numbered 0–9–0 beneath a gray main scale, with
a red/cyan arrow marking the aligned graduation used to read the fractional measurement.
This is the standard didactic diagram used across many textbooks/encyclopedias for
teaching vernier reading — exactly the "how to read the nonius" classic diagram
requested.

## enr-bearing-fest-los-lagerung.svg

**What it shows:** A labeled cross-section engineering diagram of a shaft supported by
two rolling bearings — the German "Fest-Los-Lagerung" (fixed-bearing / floating-bearing
arrangement): the left bearing is axially fixed (Festlager) and the right bearing is
allowed to float axially (Loslager), each bearing pair fixed on the shaft by circlips
per DIN 471. This is an excellent, precise match for "bearing mounted on shaft against
shoulder" as a cross-section diagram, and directly illustrates the fixed/floating
concept the task specifically asked to find.

- Original file: `File:Fest-Los-Lagerung.svg`
- Commons page: https://commons.wikimedia.org/wiki/File:Fest-Los-Lagerung.svg
- Author: Jahobr
- License: CC BY-SA 3.0
- Suggested topic: "Bearing mounted on shaft against shoulder / fixed-floating bearing
  arrangement" (topic 12)
- Type: Diagram

## Topics I could NOT find ideal images for / honest gaps

- **Granite surface plate**: only found a cast-iron scraped surface plate
  (`ch16-surface-plate.jpg`). Multiple search variants for "granite surface plate,"
  "toolmakers granite plate," "inspection granite plate" returned only geology/quarry
  photos, not metrology equipment. If a granite plate photo is a hard requirement,
  this needs a source outside Commons or a different search pass.
- **Chamfering tool / chamfered edge as a photo** (rather than a diagram): I found a
  chamfering cutter tool product photo (`File:Chamfer Cutter Tool.png`, CC0) but it
  shows the cutting tool itself, not a finished chamfered edge on a part, so I chose
  the labeled chamfer-vs-fillet diagram instead as a cleaner didactic fit. If a photo
  of an actual chamfered part edge is wanted, this remains a gap.
- **Bearing shoulder without the fixed/floating framing** (i.e. a simpler single-bearing-
  against-a-shoulder photo/diagram): not pursued once the more precise Fest-Los-Lagerung
  diagram was found, since it already shows both bearings seated against shaft shoulders
  plus circlip axial fixation in one image.
- All other requested topics (vernier caliper, vernier reading, micrometer, dial
  indicator, gauge blocks, feeler gauges, double nut, burr/deburring, circlip, bearing
  fixation) have solid, precisely-matching images as documented above.
