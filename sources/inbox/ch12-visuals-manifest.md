## ch12-welded-tube-frame.jpg

**What it shows:** A welder/frame-builder in a small shop, wearing tinted
safety glasses, brazing/TIG-welding a round-tube bicycle frame held in a
dedicated frame-building jig — torch flame visible at a tube joint, gas
bottle and jig hardware in the foreground. **Honest caveat**: this is a
*bicycle* frame, not a machine-base/table frame as the topic hints — I could
not find a well-licensed Commons photo of someone welding a square-tube
machine-base or table frame specifically. The image is included because it is
a genuine, clear demonstration of the underlying skill the topic is about
(joining tube/profile stock into a welded frame in a jig); caption it
accordingly or treat it as a stand-in until a closer match turns up.

- Original file: `File:UFV Bicycle frame building (8639724661).jpg`
- Commons page: https://commons.wikimedia.org/wiki/File:UFV_Bicycle_frame_building_(8639724661).jpg
- Author: University of the Fraser Valley (UFV), via Flickr
- License: CC BY 2.0
- Suggested topic: 4 — Welded steel/tube frame from profiles (closest
  available match — see caveat above)
- Type: Photo (small original, 640×432 — this is the largest size Commons
  has for this file)

## Gaps / honest notes

- **Topic 4 (welded steel frame from profiles / machine base or table
  frame)** was the hardest to source cleanly. Extensive searching (English,
  German, French terms; Commons category browsing under
  `Category:Welding`, `Category:Steel-profile storage`, `Category:Bicycle
  frame welding`) turned up no well-licensed photo of someone welding a
  square-tube/angle-iron machine base or workbench frame specifically. The
  bicycle-frame jig photo used instead is a legitimate substitute for
  teaching "profiles joined into a welded frame," but it is not a machine
  base — flag this if a closer match is needed later.
- **Topic 3 (stock rack)** is closest available match, but shows mostly
  round bar stock rather than square tube/angle iron specifically — the
  rack format and mixed-profile context are right, but a rack showing purely
  square tube and angle iron was not found with a compatible license.
- No suitable image was found and rejected for licensing reasons during this
  search (a couple of promising hits — e.g. `File:Metallbau.jpg`, a Commons
  "steel profile" thumbnail — turned out to be tagged "Copyrighted free use,"
  which is not on the allowed list, so it was skipped).
- All 8 topics from the brief are covered with one image each; no duplicates
  of ch07-machine-shop.jpg / ch07-welder-at-work.jpg were introduced.
